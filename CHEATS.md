# Cheat Codes

Type these codes on your keyboard at any time (menu, gameplay, pause).

| Code     | Effect              | Type        |
|----------|---------------------|-------------|
| `IDDQD`  | God Mode — invincible, HP stays full | Toggle on/off |
| `IDKFA`  | One Hit Kill — enemies die in one hit | Toggle on/off |
| `BIGXP`  | XP Boost ×10 | Toggle on/off |
| `NOCLIP` | Full Heal — instantly restores HP to max | One-shot |
| `SHOWME` | Skip Stage — jump to the next room | One-shot |
| `MAXLVL` | +10 Levels — gain 10 levels with +HP/DMG/SPD | One-shot |
| `BRUTE`  | Summon The Brute — teleports to boss arena | One-shot (in-game) |
| `WARLOCK`| Summon The Warlock — teleports to boss arena | One-shot (in-game) |
| `PHANTOM`| Summon The Phantom — teleports to boss arena | One-shot (in-game) |
| `JUGGER` | Summon The Juggernaut — teleports to boss arena | One-shot (in-game) |

Active toggle cheats show badges in the top-right corner of the screen.
